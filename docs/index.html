a minimal Neovim for C/C++ development

Features

  • Minimalist IDE experience
  • Lua-powered configuration
  • Works in headless and TUI setups

Quick Install

git clone https://github.com/zimmerman-dev/cc-neovim.git

Run Neovim and let Lazy install plugins.
